### 工作原理

- workflow 工作流
- job 任务 可以完成多个任务
- step 步骤 每个步骤是由多个 step 构成
- action

name: workflow 名称
on: 触发 workflow 条件
push pull_request

```yaml
on:
  push:
    branches:
      - master
```

### jobs

```yaml
jobs:
  my_first_job:
    name: my first job
  my_second_job:
    name: my second job
    needs: my_first_job
    runs-on: ubuntu-latest
        steps:
            name: test
            env:
                MY_VAR: hi
                FIRST_NAME: changfeng
            run:
                echo $FIRST_NAME
                npm install pnpm -g
                pnpm install
                pnpm run build
```
